# 재고 관리 매뉴얼

## 기본 원칙

### 1. 선입선출 (FIFO)
- 먼저 들어온 상품을 먼저 판매
- 유통기한이 짧은 상품일수록 철저히 관리
- 신상품 입고 시 기존 상품을 앞쪽에 진열

### 2. 안전재고 유지
- 각 상품별 안전재고 기준 준수
- 안전재고 이하 시 즉시 발주
- 품절로 인한 판매 기회 손실 방지

### 3. 유통기한 관리
- **당일 유통기한**: 도시락, 샌드위치, 햄버거
  - 매일 발주, 폐기율 최소화
  - 점심/저녁 피크 타임 전 품절 방지
  
- **단기 유통기한 (1-7일)**: 삼각김밥, 우유, 빵
  - 주 2-3회 발주
  - 유통기한 2일 전부터 할인 판매 고려
  
- **중기 유통기한 (7-30일)**: 요구르트, 주스, 치즈
  - 주 1-2회 발주
  - 선입선출 철저
  
- **장기 유통기한 (30일 이상)**: 라면, 과자, 음료
  - 월 1-2회 대량 발주 가능
  - 재고 회전율 체크

## 재고 점검 주기

### 매일
- 당일 유통기한 상품 확인
- 품절 상품 체크
- 냉장/냉동 온도 확인

### 주간
- 전체 재고 실사
- 유통기한 임박 상품 파악
- 발주 계획 수립

### 월간
- 재고 회전율 분석
- 폐기율 분석
- 발주 패턴 최적화

## 재고 회전율 관리

**목표 회전율**:
- 즉석식품: 주 5-7회
- 유제품: 주 2-3회
- 스낵/음료: 주 1-2회
- 생활용품: 월 1-2회

**회전율 개선 방법**:
- 느린 상품: 프로모션, 진열 위치 변경
- 빠른 상품: 발주량 증가, 재고 확보
