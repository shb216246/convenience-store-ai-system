# 발주 가이드

## 발주 시점 결정

### 자동 발주 기준
1. **재고 기준**: 안전재고 이하
2. **판매 트렌드**: 최근 7일 평균 판매량
3. **계절/날씨**: 예상 수요 변동
4. **이벤트**: 공휴일, 행사 등

### 수동 발주 필요 상황
- 신상품 출시
- 대규모 프로모션 계획
- 예상치 못한 수요 급증
- 공급 불안정 상품

## 발주량 계산

### 기본 공식
```
발주량 = (일평균 판매량 × 발주 주기) + 안전재고 - 현재고
```

### 조정 요소
1. **날씨 예보**
   - 비/눈: 즉석식품 +20%, 우산 +500%
   - 더위: 아이스크림 +50%, 음료 +40%
   - 추위: 라면 +30%, 커피 +20%

2. **요일 패턴**
   - 월요일: 주말 소진 보충
   - 금요일: 주말 대비 증량
   - 주말: 직장인 상품 감소, 가족 상품 증가

3. **계절 변동**
   - 여름: 냉동식품, 음료 2-3배
   - 겨울: 라면, 커피 1.5-2배
   - 환절기: 마스크, 건강식품 증가

## 발주 주기별 전략

### 매일 발주
- 도시락, 샌드위치, 햄버거
- 전날 판매량 기준
- 폐기 최소화 우선

### 주 2-3회 발주
- 삼각김밥, 우유, 빵, 샐러드
- 월/수/금 또는 화/목/토
- 유통기한 고려

### 주 1회 발주
- 라면, 과자, 음료, 스낵
- 주말 수요 고려
- 대량 발주로 단가 절감

### 월 1-2회 발주
- 생활용품, 담배
- 재고 공간 고려
- 대량 발주 할인 활용

## 발주 시 체크리스트

✅ 현재고 정확히 파악
✅ 유통기한 임박 상품 확인
✅ 최근 판매 트렌드 분석
✅ 날씨 예보 확인
✅ 다가오는 이벤트 체크
✅ 냉장/냉동 공간 확인
✅ 발주 예산 확인
